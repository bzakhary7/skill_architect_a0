# SkillArchitect

A professional Agent Zero plugin that provides automated evaluation, testing, and structured auditing of agent skills. Originally adapted from the open-source OpenCode Skill Creator, this robust tool leverages A0's native CLI bridge to seamlessly run local TS dependencies without persistent framework clutter.

## Features
- **Local Persistence Execution:** Automatically installs necessary Node `tsx` execution wrappers locally using Agent Zero's internal `hooks.py` engine.
- **Skill Optimization Loop:** Tests descriptions and tests trigger logic via evaluating query responses.
- **Agent Prompt Debugging:** Bridges native Python tests against raw test-cases.

## Usage
Simply invoke the provided skills within your Agent Zero interactive shell to launch the CLI interface tools.
